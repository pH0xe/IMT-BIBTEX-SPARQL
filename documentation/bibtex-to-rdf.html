Admins

Up 
delete /api/bibtex/{id}
Delete a bibtex file (deleteFileEndpoint)
Delete a bibtex file from database and history.

Path parameters

id (required)
Path Parameter — Id of the bibtex file

Return type

DeleteResponse

Example data

Content-Type: application/json
{
  "success" : true,
  "error" : "File not found"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successfully deleted bibtex file DeleteResponse

400

Error when deleting file DeleteResponse

500

Error with database Error

post /api/bibtex/{id}
Restore a bibtex file (restoreFile)
Restore a bibtex file from history.

Path parameters

id (required)
Path Parameter — Id of the bibtex file

Return type

ConvertResponse

Example data

Content-Type: application/json
{
  "warnings" : [ "Warning message", "Warning message" ],
  "message" : "Invalid input"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successfully restored bibtex file ConvertResponse

418

Bibtex file contains syntax errors ConvertResponse

500

Error with database Error

503

Error with fuseki server Error
Up 
post /api/bibtex
Upload a bibtex file (uploadFile)
Upload a bibtex file in database, parse it and send it to fuseki server.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • multipart/form-data

Form parameters

file (optional)
Form Parameter — format: binary

Return type

ConvertResponse

Example data

Content-Type: application/json
{
  "warnings" : [ "Warning message", "Warning message" ],
  "message" : "Invalid input"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successfully uploaded bibtex file ConvertResponse

400

Invalid file uploaded Error

418

Bibtex file contains syntax errors ConvertResponse

500

Error with database Error

503

Error with fuseki server Error
